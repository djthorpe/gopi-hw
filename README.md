# gopi-hw

This respository contains hardware implementations for
[gopi](http://github.com/djthorpe/gopi)
and some example programs in the `cmd` folder.  The gopi modules 
provided by this repository are:

| Platform | Import | Type | Name |
| -------- | ------ | ---- | ---- |
| Raspberry Pi | `github.com/djthorpe/gopi-hw/sys/hw` | `gopi.MODULE_TYPE_HARDWARE` | `sys/hw/rpi` |
| Darwin | `github.com/djthorpe/gopi-hw/sys/hw` | `gopi.MODULE_TYPE_HARDWARE` | `sys/hw/darwin` |
| Linux | `github.com/djthorpe/gopi-hw/sys/metrics` | `gopi.MODULE_TYPE_OTHER` | `sys/metrics/linux` |
| Darwin | `github.com/djthorpe/gopi-hw/sys/metrics` | `gopi.MODULE_TYPE_OTHER` | `sys/metrics/darwin` |
